# 🛡️ Linux SOC Labs & CyberSecurity Practice

¡Bienvenido a mi repositorio de entrenamiento en Ciberseguridad! Este espacio está dedicado a centralizar mis laboratorios prácticos, simulaciones de defensa y comandos de auditoría en sistemas operativos Linux, enfocado en el rol de **Analista de SOC / Incident Response**.

## 🎯 Objetivos del Repositorio
* Documentar la resolución de incidentes tácticos en entornos reales.
* Desarrollar habilidades sólidas en la terminal de Linux para auditorías forenses.
* Crear una guía de referencia rápida para monitoreo de infraestructura y redes.

## 📁 Estructura de los Módulos Desarrollados

### 🗺️ Módulo 1: Orientación y Navegación del Sistema
* Comandos esenciales de reconocimiento de entorno (`pwd`, `ls -la`).
* Identificación de archivos ocultos y metadatos básicos de seguridad.

### 📁 Módulo 2: Gestión de Permisos y Filtros Forenses
* Aislamiento estricto de archivos confidenciales (`chmod 700`).
* Uso de cañerías (`|`) y filtrado avanzado de logs/evidencias con `grep`.

### 💾 Módulo 3: Monitoreo Local y Control de Procesos
* Análisis en tiempo real de recursos y telemetría de rendimiento (`top`, `df -h`).
* Mitigación inmediata y contención de procesos colgados o maliciosos (`kill -9`).
* Auditoría de registros de autenticación (`/var/log/auth.log`).

### 📡 Módulo 4: Redes, Puertos y Conectividad Externa
* Mapeo perimetral y auditoría de sockets en estado de escucha (`ss -tulnp`).
* Análisis de direccionamiento privado (`ip a`) e investigación de infraestructura externa (`ping`, `nslookup`, `traceroute`).
